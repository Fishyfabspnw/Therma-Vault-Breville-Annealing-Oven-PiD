# ThermaVault PID — Bill of Materials

> BOM is actively changing as the project evolves.

## Core Electronics

| Component | Model / Type | Qty | Notes |
|---|---|---:|---|
| PID Controller | Novus N20K48 | 1 | Main process controller with ramp/soak support |
| Solid State Relay | Celduc SO943460 40A | 1 | Main heater switching SSR |
| SSR Heat Sink | Generic aluminum SSR heat sink | 1 | Required for thermal management |
| Temperature Sensor | PT100 RTD | 1 | Chamber temperature sensing |
| Main Power Switch | Illuminated rocker switch | 1 | Main system power |
| Fuse / Breaker | Panel-mount protection device | 1 | Required line protection |
| Terminal Blocks | WAGO / terminal strip | As needed | Power distribution |
